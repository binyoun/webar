experiment 04 · score

Sonify: text as a music box strip

lineage: Vanishing Waters / Perforated Atlas (memory → perforation → sound and light)

This is a deterministic translation: every character becomes a note by a fixed rule (its character code, modulo the scale length). No randomness, no AI. The same sentence always produces the same melody, the way the same perforated strip always plays the same tune. Determinism is the point: the memory owns its sound.

each column is one character · height is pitch · spaces are rests

experiments to try
  1. Change osc.type to 'square' or 'triangle'. How does timbre change what the memory feels like?
  2. Type the same sentence in Vietnamese with and without diacritics. The tone marks change the character codes, so dấu literally changes the melody. What does that say about romanization?
  3. Replace the scale array with two notes only. When does a translation become an erasure?
  4. Attach your melody to your AR work: add an audio field to your config entry with your sentence as source.
