Lenses: one image, many machines

lineage: SELECT PERCEPTION ENGINE (six fictional AI engines reading one Hwajeopdo painting)

The image never changes. Only the reading changes. Each "engine" is a fiction: a named point of view with a color and a voice. The lesson hiding inside the joke is real: every actual machine-vision system is also a point of view wearing the costume of objectivity.
